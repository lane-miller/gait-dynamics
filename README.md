# gait-dynamics
Python pipeline for gait and running dynamics analysis from IMU sensor data — signal processing, ML, automated testing.

# Raw Data

This directory is not tracked by git and must be populated manually.

## Dataset: Camargo et al. (2021)

Multi-modal lower-limb biomechanics dataset collected during walking,
running, and stair tasks. Includes IMU, EMG, and ground reaction force data.

**Source:** PhysioNet  
**Citation:** Camargo J, Ramanathan A, Flanagan W, Young A (2021). A comprehensive, 
open-source dataset of lower limb biomechanics in multiple conditions of stairs, 
ramps, and level-ground ambulation and transition. Journal of Biomechanics.

## Download Instructions

1. Visit https://physionet.org/content/lower-limb-biomechanics/1.0.0/
2. Create a free PhysioNet account if you don't have one
3. Download and place contents in this directory
